## Features
- **Military-Grade Security**: AES-256-GCM encryption for all items.
- **Batch Processing**: Process multiple items at once with infinite scaling.
- **Multi-Threaded**: Utilizes Web Worker pool for parallel compression and encryption.
- **Self-Extracting**: Files are packed into a universally accessible HTML format.

## Security & Integrity
HDA Vault uses the **HDA Protocol v3.0**, featuring:
- **AES-256-GCM Encryption**: Secure, local-only key derivation.
- **SHA-256 Verification**: Cellular integrity checks for all data.
- **24-bit Virtual Book Logic**: Pixel-perfect data mapping.
